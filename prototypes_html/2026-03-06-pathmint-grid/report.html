PathMint Grid · Reporte

Juego de habilidad/destreza con movimiento en grid, trampas, monedas, timer y 5 niveles diarios.

Mecánica

  • Control por teclado y botones touch.
  • Recoger monedas mejora score y tiempo.
  • Evitar trampas y llegar a meta para avanzar.

Loop diario

  • Run compacta de 2–5 min.
  • Dificultad incremental por nivel.
  • Badge PathMint Champion como recompensa de mastery.

KPI objetivo

  • Completion run >40%
  • D1 retention >33%
  • Average coins por run >=7
