Sum Tiles 24 — Detailed report

Date: 2026-03-06 · Playable prototype

Strategy

  • Core loop: 60–120s sessions, daily seed, visible streak progress.
  • Audience: casual puzzle players with short-session habit.
  • Differentiation: readable UI, low friction, daily mastery ladder.

SEO / ASO / SEM

  • ASO title: "Sum Tiles 24: Daily Puzzle" + subtitle with retention hook.
  • Keyword clusters: daily puzzle, brain challenge, offline game, quick game.
  • Store assets: first screenshot = loop in 3 taps, second = reward, third = leaderboard.
  • SEM: low-CPI UAC split by intent (daily / brain / offline), kill losers <72h.

Loop + Rewards

  • Entry: open → daily board → solve → reward chest.
  • Rewards: coins, streak badges, weekly modifiers, cosmetic unlocks.
  • Social mechanic: postable score card with challenge CTA.

KPIs

  • Activation >82%, first puzzle completion >70%.
  • Retention goals: D1 42%+, D7 16%+, D30 6%+.
  • Monetization: rewarded ad opt-in 28–35%, IAP 1.5–2.2%.
  • Quality: crash-free sessions >99.4%, ANR <0.35%.

Experiments

  1. Difficulty ramp A/B for first 3 sessions.
  2. Reward cadence A/B (daily vs 48h).
  3. Creative hook A/B (7s vs 12s TikTok-style).

Risks

  • Novelty decay in <2 weeks → mitigate with weekly themed packs.
  • Paid CPI volatility → hedge with organic creator + Reddit loop.
  • Store confusion → explicit first-screen value prop text.
