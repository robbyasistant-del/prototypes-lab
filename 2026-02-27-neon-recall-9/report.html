Neon Recall 9 — Quality Prototype Report

Fecha: 2026-02-27 · Tipo: Memory / Reflex / Daily skill challenge

Core loop

  1. Start → se muestra mapping 1..9 aleatorio en grid 3x3 durante 3s.
  2. Se oculta todo y jugador debe pulsar casillas en orden 1→9.
  3. Acierto: casilla verde + número visible.
  4. Fallo: reset de progreso + penalización de 5s.
  5. Win al completar 9; Lose si tiempo >= 60s agotado.

Retention / replay hooks

  • Run corto (<60s): fricción baja, alta repetición.
  • Penalización tangible (-5s) genera tensión y mejora de habilidad.
  • Variación diaria por random seed permite rutina tipo streak.

ASO/SEO angle

Keywords candidatas: memory game, brain training, daily challenge, speed puzzle, focus game, 3x3 memory.

KPI sugeridos

  • Completion rate (Win %) por sesión.
  • Median completion time (solo wins).
  • Retries per session y drop-off tras primer fallo.
  • D1 return para usuarios que juegan >=3 runs primer día.

Riesgos

  • Dificultad percibida alta para nuevos usuarios (onboarding mínimo).
  • Accesibilidad visual en pantallas pequeñas (tamaño de celda/número).
  • Necesidad de audio/haptic feedback para pulido de sensación premium.
