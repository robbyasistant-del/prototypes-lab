Guess the Mon Daily — Prototype Report

Fecha: 2026-02-28 · Categoría: Silhouette Quiz + Trivia Speedrun

Propuesta

Daily run de 8 rondas orientado a nostalgia fan + share loop. El usuario reconoce siluetas y resuelve micro-trivia de tipo/región bajo presión de tiempo.

Core Loop

  1. Inicio de run (90s).
  2. Ronda: silueta oscura + pregunta (name o trivia).
  3. Feedback instantáneo (reveal, score, streak, penalización).
  4. Final de run con score y share card.

SEO/ASO/SEM

Keywords: daily monster quiz, silhouette game, trivia speedrun, fandom challenge, nostalgia quiz.

Creatives: hook en 2s con silueta + timer + reveal del nombre.

KPIs objetivo

  • Run completion rate (8/8)
  • Average score por sesión
  • Share card copy rate
  • D1 retorno usuarios con streak >3

Riesgos y mitigación

  • IP/legal: usar versión original sin assets oficiales para test de mecánica.
  • Dificultad: balancear penalización y bonus de velocidad.
  • Fatiga: rotar banco de preguntas diariamente.
